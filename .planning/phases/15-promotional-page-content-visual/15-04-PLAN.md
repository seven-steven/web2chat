| verification commands → executor workflow | 缺少统一验证闭环会让 truth regressions 溜过 Phase 15 |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-15-10 | Repudiation | `verify-build.mjs` | mitigate | 把 final-page 关键标记写入 smoke assertions，失败时输出明确错误，避免“build 成功但页面错误”无法追责 |
| T-15-11 | Tampering | README install target | mitigate | 使用稳定安装 heading/anchor，并通过 `pnpm verify:readme` 验证链接目标存在 |

| T-15-SC | Tampering | npm/pnpm installs | accept | 本计划不新增依赖、不运行 package-manager install，供应链面保持不变 |
