<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| docs truth → marketing data | 规划事实源被转写为可发布页面文案，任何越界 wording 都会变成公开误导 |
| locale data → UI render | 缺键或错键会把内部 key 名直接暴露到页面 |
| CTA config → external navigation | 错误链接会把用户带到错误的仓库入口或安装路径 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-15-01 | Spoofing | `site-content.ts` platform data | mitigate | 仅返回 OpenClaw/Discord/Slack/Telegram，并在测试中断言顺序与 Telegram 风险标签，防止把未 shipped 平台伪装为已支持 |
| T-15-02 | Tampering | locale JSON permission copy | mitigate | permission facts 只从 production `wxt.config.ts` 可验证事实落地，测试中明确禁止 `tabs` 与 static `<all_urls>` production claims |
| T-15-03 | Information Disclosure | trust copy | mitigate | privacy facts 只使用 `PRIVACY.md` 中已公开的 local-first/no-telemetry 机制，不添加未验证内部实现细节 |
| T-15-SC | Tampering | npm/pnpm installs | accept | 本计划不新增依赖、不运行 package-manager install，供应链面保持不变 |
</threat_model>
