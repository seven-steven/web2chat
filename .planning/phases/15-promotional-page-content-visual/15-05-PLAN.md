<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CTA primitive → external navigation | 错误或不完整的 anchor 语义会让用户看见按钮但无法稳定离开当前页 |
| marketing-only class composition → shipped truth UI | 视觉修正若越界到 token 层，可能误伤 popup/options 已建立的设计契约 |
| automated DOM checks → real browser behavior | 单测能覆盖 href/semantics，但真实点击落点仍需要浏览器复验 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-15-G1 | Spoofing | `apps/marketing/src/components/cta-button.tsx` | mitigate | 通过共享 CTA primitive 强制外链语义、稳定测试钩子与三处 CTA 统一回归，防止出现“看似按钮正常、实际一个实例失效”的假象 |
| T-15-G2 | Tampering | `apps/marketing/src/app.tsx` local marketing styling | mitigate | 只允许在 marketing 组件层局部收敛 Hero/card/CTA band 样式，禁止改动 `shared/styles/design-tokens.css`，避免误伤 extension UI |
| T-15-G3 | Repudiation | CTA/UAT verification chain | mitigate | 先用 DOM 回归锁住 href/target/rel/data-testid，再用 blocking human verify 记录真实浏览器点击结果，避免“代码看起来对但没人确认真机可用” |
| T-15-SC | Tampering | npm/pnpm installs | accept | 本计划不新增依赖、不运行 package-manager install，供应链面保持不变 |
</threat_model>
