## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-02-01 | Spoofing | `isLoggedOutUrlForAdapter` | mitigate | Parse with `new URL`, require adapter host match, and compare only pathname patterns per D-115. |
| T-09-02-02 | Tampering | path pattern syntax | mitigate | Support only exact string and trailing `*` prefix semantics; no RegExp/URLPattern per D-115. |
| T-09-02-03 | Information Disclosure | OpenClaw/self-hosted redirects | mitigate | Unconfigured platforms return false so arbitrary host/path shape is not misclassified as login per D-117. |
| T-09-02-04 | Repudiation | multiple login remap branches | mitigate | Route all pipeline remaps through one helper per D-118 and test tab complete + SPA paths. |
</threat_model>
