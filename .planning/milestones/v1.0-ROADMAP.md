## Overview

Chrome MV3 web-clipper 扩展。抓取页面结构化元数据（title / url / description / create_at / content），与用户绑定的 prompt 一起，通过 content script DOM 注入投递到目标 IM 或 AI Agent web 聊天会话中。v1.0 覆盖 OpenClaw Web UI + Discord 两条投递通道，47/47 v1 需求全部完成。

## Key Decisions

| Decision | Rationale | Outcome |
|----------|-----------|---------|
| Chrome MV3 only | 用户首发只覆盖 Chrome | ✓ 全程无兼容性阻塞 |
| MVP 仅 OpenClaw + Discord | URL pattern 确定，优先打通主链路 | ✓ IMAdapter 接口可复用 |
| 新开 tab + content script 注入 | 不需要服务端 token / OAuth | ✓ 两种注入路径均稳定 |
| 所有配置 chrome.storage.local | 隐私优先 + 单设备 | ✓ 全部 typed repo |
| optional_host_permissions 动态申请 | 自部署 origin 不可枚举 | ✓ Phase 4 UAT 通过 |
