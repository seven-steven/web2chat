## Overview

web2chat 在 v1.2 交付了一个仓库内可独立构建的静态 web 宣传页（`apps/marketing`），用于对外介绍产品定位、支持平台、核心流程、隐私承诺与安装入口。整个 milestone **不改扩展运行时主链路**——所有工作落在新增的 marketing workspace app、共享 design tokens 边界、claims/privacy 跨源校验脚本与发布验收基线上。

核心成果是一条「source-first → artifact-second → page-last」的防过宣传机制：宣传页所有对外 claim 都能追溯到 `PROJECT.md` / `PRIVACY.md` / `STORE-LISTING.md` / 生产 `wxt.config.ts`，并由 `verify:claims` 脚本在 CI 里自动拦截任何漂移。诚实边界（Telegram live UAT pending、Phase 11/12 Nyquist partial、Feishu/Lark dropped）被刻意保留在风险说明与 release notes 中，而非主卖点。

## Key Accomplishments

1. 建立仓库内独立 marketing workspace app（`apps/marketing`），宣传页可独立 build / preview / smoke test，且通过 BUILD-03 import 隔离测试证明不引入扩展 runtime 模块（service worker、storage repo、messaging、permissions、adapters）。
2. 实现双语（en / zh_CN）8-section 静态宣传页，覆盖 Hero、use cases、structured-payload 示例、支持平台、三步核心流程、隐私/权限、已知限制、CTA，en/zh_CN locale key 100% 同构。
3. 交付 `verify:claims` 跨源一致性校验器（5 条规则，禁止 cloud sync / 用户行为分析等过宣传词，引用 `shipped-platforms.json` 单一事实源），并作为 CI gate 自动拦截任何 claim 漂移。
4. 完成 WCAG G201 外链可访问性：3 个外链 CTA 可见 ↗ glyph + `.sr-only` 新标签页警告（标准 WCAG boilerplate：1px box + clip rect + clip-path inset），由 Playwright 实测 zh_CN/en 双 locale 确认。
5. 建立发布验收运营基线：根级 `MAINTENANCE.md` 给出 source-first → artifact-second → page-last 维护链，`CHANGELOG [v1.2]` 含 3 项诚实 Known Issues，CI 单 job 接入 4 个 marketing/claims gate（含 load-bearing `verify:manifest` → `verify:claims` 顺序不变量）。
6. 关闭宣传页全部人工 UAT 维度：G201 可见字形 + responsive 无溢出经 Playwright `launchPersistentContext` 实测验证（375px/1280px 双断点），Phase 15/16 verification status 从 `human_needed` 推到 `passed`。

## Key Decisions

| Decision | Rationale | Outcome |
|----------|-----------|---------|
| 仓库内静态宣传页（非独立站点） | shipped platform set 已具对外说明价值；随扩展同仓维护避免引入独立站点运维 | ✓ marketing app 与扩展共仓，零额外部署 |
| 共享 design tokens 而非共享组件 | extension popup 与 marketing 需视觉一致但运行时隔离；仅共享 `shared/styles/` CSS，禁止 import 扩展 runtime | ✓ BUILD-03 隔离测试持续通过 |
| `verify:claims` 作为 self-enforcing CI gate | 人工 checklist 易腐化；把 claim 一致性变成编译期拦截 | ✓ 5 条规则在 CI 每个 PR 自动跑 |
| WCAG G201（visible glyph + sr-only）而非 aria-label | 保留可访问名来自字符串子节点，sr-only 走 i18n 路由 | ✓ Playwright 实测双 locale glyph 可见 + sr-only 视觉隐藏 |
| CI 单 job 扩展（非并行 jobs） | 4 个新 gate 各 ~sub-15s，并入现有 verify job 比并行省 checkout+install | ✓ 8→12 run steps，wall-clock 仅 +10-15s |
| 诚实 Known Issues 写入 release notes | Telegram UAT / Nyquist partial / Feishu-Lark dropped 不应被宣传页掩盖 | ✓ CHANGELOG [v1.2] 含 3 项边界，TRUST-03 闭合 |
