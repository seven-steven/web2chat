# Quick Task 260616-235517: webpage 添加 favicon - Plan

## Goal

为 `apps/marketing` 的 web 宣传页面添加 favicon，使浏览器标签页和构建产物都能引用项目现有 Web2Chat 图标。

## Tasks

1. 复用现有项目图标作为 marketing favicon
   - 文件：`apps/marketing/public/favicon.svg`
   - 动作：从 `public/icon/icon.svg` 复制 SVG 图标到 Vite 静态资源目录。
   - 验证：确认构建后 `dist/favicon.svg` 存在。

2. 在 HTML head 中声明 favicon
   - 文件：`apps/marketing/index.html`
   - 动作：添加 `<link rel="icon" type="image/svg+xml" href="/favicon.svg" />`。
   - 验证：确认构建后 `dist/index.html` 保留 favicon link。

3. 运行可自行执行的验证
   - 命令：`npm run build --workspace apps/marketing`（如 workspace 不适用则在 `apps/marketing` 下运行 build）和 `npm run verify:build`。
   - 完成条件：构建与静态校验通过。
