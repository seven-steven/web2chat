# Quick Task 260509: GitHub Actions CI/Release + 安装文档

## 背景

项目已有 `.github/workflows/ci.yml`（push/PR 触发 typecheck + lint + test + verify:manifest），但没有 release workflow。需要：
1. 添加 tag-triggered release workflow，构建生产版本并发布 zip 制品
2. README 中添加从 GitHub Releases 下载安装的说明

## Tasks

### Task 1: 创建 Release workflow

- **action:** 创建 `.github/workflows/release.yml`
  - 触发条件：push tag `v*`
  - Job: `release`
    - checkout + pnpm + node 20
    - `pnpm install --frozen-lockfile`
    - `pnpm typecheck && pnpm lint && pnpm test && pnpm verify:manifest`
    - `pnpm zip`（WXT 生成 `.output/web2chat-*.zip`）
    - 使用 `softprops/action-gh-release` 创建 GitHub Release，上传 zip
- **verify:** workflow YAML 语法正确，文件存在

### Task 2: 更新 README 添加 Release 安装说明

- **action:** 在 README.md 安装章节添加 "从 GitHub Releases 下载" 子章节
  - 说明：前往 Releases 页面 → 下载最新 zip → 解压 → chrome://extensions 加载已解压扩展
- **verify:** README 中的安装说明清晰完整
