# Plan: Settings Back Navigation

## Goal

当 popup 处于 settings 视图时，提供明确的"返回主页"交互。当前齿轮图标是 toggle，但视觉上不直观。

## Approach

将 PopupChrome 的齿轮图标在 settings 模式下替换为左箭头（←）图标，配合 tooltip 文案切换，给用户清晰的"返回"信号。

## Tasks

### Task 1: Settings 模式下齿轮替换为返回箭头

**files:**
- `entrypoints/popup/components/PopupChrome.tsx`
- `entrypoints/popup/App.tsx`

**action:**
1. PopupChrome: 当 `showSettings=true` 时，将齿轮 SVG 替换为 chevron-left 箭头 SVG
2. 箭头图标使用与齿轮相同的 24x24 viewBox、stroke-width 1.75
3. `aria-label` 和 `title` 根据 showSettings 状态切换：settings 时显示"返回"文案，send 时显示"设置"文案
4. 去掉 settings 模式下的 rotate-[60deg] 旋转效果（箭头不需要旋转）
5. 保持现有的 hover 动效

**verify:** 齿轮点击进入 settings 后，图标变为左箭头，点击箭头返回 send 模式

**done:** Settings 模式有明确的返回箭头图标

### Task 2: i18n — 添加返回按钮 tooltip 文案

**files:**
- `locales/en.yml`
- `locales/zh_CN.yml`

**action:**
1. 添加 `popup_chrome_back_tooltip` 键，en = "Back"，zh_CN = "返回"
2. PopupChrome 根据 showSettings 选择 tooltip key

**verify:** 两种语言下 tooltip 文案正确

**done:** i18n 覆盖完整
