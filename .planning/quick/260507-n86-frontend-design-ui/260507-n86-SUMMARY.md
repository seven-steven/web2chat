## 设计语言摘要 (DESIGN.md 9 节)
- **Visual Theme:** Editorial / data-dense（paper canvas, ink-on-paper hierarchy, rust accent）
- **Color:** stone neutrals + 单一 rust 强调（`#c2410c` light / `#fb923c` dark），抛弃 sky-blue cliché
- **Typography:** 三层字型 — `font-serif` (display) + `font-sans` (body) + `font-mono` (data)，全部系统字体栈，0 KB
- **Components:** Edge-line cards、margin-note banners、sidenote hints、footnote markers、host chips、dispatch pills、UPPERCASE 11px labels
- **Layout:** popup 360px / options 720px 单列；`{4,8,12,16,20,24,32}` spacing scale
- **Depth:** 边线 > 阴影；shadow-pop 仅 modal/dropdown
- **Motion:** stagger reveal、press-style buttons、1.6s editorial spinner；全部 `prefers-reduced-motion` guard
- **Anti-cliche:** stone（暖）替代 slate（冷），rust 替代 sky，serif display 替代单一 sans，UPPERCASE 11px 标签，asterism 装饰，footnote ¹ marker
